# 🌐 Tecnología General

Pagina web sobre tecnología

## 📖 Descripción del Proyecto
Este proyecto es una página web moderna e informativa sobre tecnología en general. Exploramos temas fascinantes como las tendencias actuales (Inteligencia Artificial, 5G/6G, Realidad Aumentada), un recorrido por la historia de la tecnología y un análisis profundo sobre su impacto en la sociedad. Proyecto desarrollado por estudiantes de la Universidad San Francisco de Quito (USFQ) y ESPE.

## 💻 Tecnologías Utilizadas
* **HTML5:** Estructura semántica de la página.
* **CSS3:** Diseño moderno y responsivo.
* **JavaScript:** Interactividad y dinamismo.

## 🚀 Instrucciones de Uso
Para ver este proyecto en tu computadora local, sigue estos sencillos pasos:
1. Abre tu terminal o consola.
2. Clona este repositorio ejecutando: `git clone https://github.com/Alexder3006/tecnologia-general.git`
3. Entra a la carpeta del proyecto: `cd tecnologia-general`
4. Abre el archivo `index.html` en tu navegador web preferido para visualizar la página.

## 🔄 Flujo de Trabajo en GitHub
Para el desarrollo de este proyecto, utilizamos un flujo de trabajo colaborativo y ordenado:
1. Cada integrante trabajó en su propia rama específica (`feature/estructura-base`, `feature/contenido-tecnologia`, `feature/estilos-interactividad`).
2. Se realizaron múltiples *commits* con mensajes claros y descriptivos para guardar el progreso de cada tarea.
3. La integración del código a la rama `main` se hizo exclusivamente a través de *Pull Requests*.
4. Cada *Pull Request* fue revisado y comentado por los miembros del equipo antes de realizar el *merge*.
